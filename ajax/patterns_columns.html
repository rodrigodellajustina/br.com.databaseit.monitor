Prefixo de Colunas
Identificação Finalidade Tipo de Dados Exemplo
NR Prefixo de coluna que irá armazenar valores numéricos inteiros/string, para conteúdos significativos. Numérico/String NR_CST
IE Indicam situações especiais, valores que demonstram status. Exemplo: 01 – Aberto, 02 – Fechado, 03 – Cancelado Numérico/String IE_STATUS, IE_SITUACAO
DS Prefixo de coluna que irá armazenar valores “string”, ou seja, caracteres descritivos. String (Caracteres) DS_OBSERVACAO, DS_NOME, DS_PRODUTO
DT Prefixo de coluna que irá armazenar valores do tipo data Data DT_LANCAMENTO, DT_EMISSAO
HR Prefixo de coluna que irá armazenar valores do tipo hora Hora HR_EMISSAO, HR_LANCAMENTO
DH Prefixo de coluna que irá armazenar valores do tipo Data/Hora Hora/Hora DH_EMISSAO, DH_LANCAMENTO
VL Prefixo de coluna que irá armazenar valores numéricos, ou seja,números que podem possuir casas decimais Numérico VL_ICMS, VL_VENDA, VL_BASECALCULO, VL_ISS
ID Prefixo de coluna que irá armazenar valores numéricos inteiros (ID, CHAVES, CHAVES ESTRANGEIRAS) Numérico ID_VENDA, ID_PEDIDO, ID_PESSOA
QT Prefixo de coluna que irá armazenar quantidades numéricas Numérico QT_ESTOQUE, QT_CONTATO, QT_VENDA, QT_COMPRA
SG Prefixo de coluna que irá armazenar valores de Siglas String SG_IMPOSTO, SG_ESTADO
XL Prefixo de coluna que irá armazenar valores contidos em colunas com especificações XML XML XML_NFE, XML_NFCE
PE Prefixo de coluna que irá armazenar valores percentuais Númerico PE_ACRESCIMO, PE_PARTICIPACAO, PE_DESCONTO, PE_UTILIZADO
IM Prefixo de coluna que ira armazenar imagens. Blob IM_FOTO_CLIENTE
FG Prefixo de coluna que irá armazenar situação de campos flag. 0 – Desligado 1 – Ligado Númerico FG_INATIVO, FG_ATIVO
JS Prefixo de coluna que irá armazenar valores padrão JSON Json Type JS_RESTWEBSERVIE, JS_INFORMATION
