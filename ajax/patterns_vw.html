Nomenclatura View

View possui um prefixo padrão de 2 dígitos (VW)

    Exemplo 1


  1. Exemplo de uma view vendas com detalhes a nível de produto

    2. vw + objetivo (vw_vendas_produto)



    Exemplo 2


  1. Exemplo de uma view compras com detalhes a nível de produto

    2. vw+ _ + objetivo (vw_compra_produto)



    Exemplo 3


  1. Exemplo de uma view produtos com estoque negativo

    2. vw+ _ + objetivo (vw_produto_estoque_negativo)



A view deverá ser desenvolvida apenas para extrair complexidade do SQL e reúso por parte dos aplicativos
